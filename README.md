# 自己紹介ページ

プログラミング学習の一環として作成した、自己紹介用の静的Webページです。

---

## 概要

名前・趣味・スキルをカード形式で表示するシンプルな自己紹介ページです。
フェードインアニメーションやスキルバーなど、CSSのみで動きを表現しています。

## 使用技術

| 技術 | 用途 |
|------|------|
| HTML | ページ構造の作成 |
| CSS | スタイリング・アニメーション |

## 主な機能

- グラデーション背景
- カードのフェードインアニメーション
- スキルバー（伸びるアニメーション付き）
- ホバーエフェクト付きリスト
